Checked on 2026-03-25.

Use this file only as a starting map. Always browse the live page before answering because Anthropic docs and redirects can change.

## Domain map

- `docs.anthropic.com`
  Main entry domain. Many Claude Code and Agent SDK pages start here.
- `code.claude.com/docs/en`
  Current canonical destination for many Claude Code docs after redirects.
- `platform.claude.com/docs/en`
  Current canonical destination for Agent SDK, API docs, and platform release notes after redirects.
